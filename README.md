# 🚀 Flutter Web Portfolio

## 📱 **Project Overview**
A responsive personal portfolio website built with Flutter Web showcasing projects, skills, and contact form.

## ✨ **Key Features**
- 📱 **Fully Responsive** - Works on all devices
- ✨ **Smooth Animations** - Fade & slide effects
- 📧 **Contact Form** - Formspree integration
- 🏗️ **Project Showcase** - Detailed project cards
- 📊 **Skills Visualization** - Animated progress bars

## 🛠️ **Tech Stack**
- **Flutter Web** - Frontend framework
- **Dart** - Programming language
- **Formspree** - Contact form backend
- **Custom Animations** - For enhanced UX

## 📁 **Project Structure**
\\\
lib/
├── main.dart           # App entry & navigation
├── sections/           # All page sections
├── utils/              # Responsive utilities
├── widgets/            # Reusable components
├── constants/          # Colors & styles
└── animations/         # Custom animations
\\\
